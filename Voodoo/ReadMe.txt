Client to send pakcets to server and recieve packets of other players and diplsay them onscreen

How to run:

	Double click the exe in Run_Windows_32
	If you want to change the port or server ip you'll need to edit the defines STARTING_SERVER_IP & STARTING_SERVER_PORT
	If you wish to run multiple clients and have them show up on server with different colors, will need to edit PLAYER_ID 

	W/Up arrow - move local player up
	S/Down arrow - move local player down
	A/Left arrow - move local player left
	D/Right arrow - move local player right
